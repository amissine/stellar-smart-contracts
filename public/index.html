This is my attempt to provide for maintenance of Stellar Smart Contracts. It is based on Tyler van der Hoeven's Stellar Turrets. In a nutshell, a Stellar Smart Contract (we call it a txFunction) runs on a signing server called Stellar Turret. The turret provides for trust-free signing of transactions it processes. The turret signs the supplied transaction only if its txFunction is ok with it.

This implementation assigns a unique key to each txFunction / turret it maintains. Using the UI below, you can:

  • upload your txFunction to the registry and receive its key;
  • list all registered txFunctions / turrets by their keys;
  • get info on a txFunction / turret selected from the list;
  • delete a txFunction from the registry - if you own it.
