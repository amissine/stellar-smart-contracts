Stellar Smart Contracts are called txFunctions. A txFunction runs on a signing server called Stellar Turret. Stellar Turrets provide for trust-free signing of transactions they process. A Turret signs the supplied transaction only if the txFunction is ok with it.

This implementation assigns a unique key to each txFunction it maintains. Using the UI below, you can:

  • upload your txFunction to the registry and receive its key;
  • list all registered txFunctions by their keys;
  • get info on a txFunction selected from the list;
  • delete a txFunction if you own it.
